# GradebookPro (Homework Clean v7)

This repo contains a split (GitHub-friendly) version of the GradebookPro HTML app.

## Structure
- `index.html`
- `assets/css/styles.css`
- `assets/js/app.js`

## Run locally
Just open `index.html` in a browser.

## Notes
- This is a static HTML app (no build step).
- Keep the Apps Script URL/secret exactly as in your original app settings.
